# Calorie Counter

A full stack calorie tracking web app built with Python, SQLite, and JavaScript. Users are able to track what they eat for each meal throughout the day, track their calorie intake against their goal, and even keep track of their weight and goals.

## Stack
- Backend: Python, Flask
- Database: SQLite
- Frontend: HTML, CSS, JavaScript

# Features
- Log foods for the specific meals (Breakfast, Lunch, Dinner, Snacks)
- Track nutritional values (Carbs, Fats, Protein)
- Track and set daily calories
- Track goals, notes, and weight

# How to run locally:
1. Clone the repository
2. Install dependencies:
    pip install flask flask-cors
3. Run the backend:
    python3 calorie_app.py
4. Open `calorie_frontend/calorie_counter.html` with Live Server in VSCode
5. Navigate to `http://127.0.0.1:5500` in your browser
